二 字符串 ---String 是由字符组成的只读数组 操作数组的函数中，不改变数组本身的api一般都可以使用（concat slice） 不改变数组本身的api都不可以使用 (splice,sort) 2.API 2.1 大小写转换 toUpperCase() 大写 toLowerCase() 小写 2.2 编码 1) charCodeAt 返回值是十进制 2) unicode编码: 字符串中 \u是表示一个unicode编码的字符，\u后面的四位表示一个字符 一般使用16进制编码 console.log("吕".charCodeAt()); // 用十进制表示: 21525 console.log("吕".charCodeAt().toString(2)); //用二进制表示: 101010000010101 console.log('吕'.charCodeAt().toString(16)); //用十六进制表示: 5415 console.log('吕'.charCodeAt().toString(8)); //用八进制表示: 52025 console.log('\u5415'); //吕 3) fromCharCode(unicode的十进制) 2.3 截取 slice(start,end) substring(start,end) 用法一样，只是substring不支持负数 substr(start,n) n 表示截取的个数 2.4 分隔 split(分隔符) 返回一个数组 如果分隔符是空字符串,那么会把字符串中的每个字符作为元素组成一个新的字符数组返回 2.5 查找关键字 indexOf(关键字,from) 返回值是关键字第一次出现的位置,如果查找不到返回-1 from是本次查到开始的位置 2.6 去除首尾不可见的字符 trim() 3.值类型和引用类型 值类型 Number String Boolean Undefined 值类型是赋值一份数据给新变量 引用类型 Array Object Function 引用类型是把自身的地址给新变量
